# docx_templating

Simple python docx templating

## Introduction

Simple module for replacing {{tags}} inside Microsoft Word docx documents

### Prerequisites

* Python 3
* python-docx >=0.8.8
* lxml >= 2.3.2

### Getting Started

Just import *docxt* inside you program. See [example.py](example.py) file.
